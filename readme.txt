== Description ==


This pluging will added nice and easy scroll .you can change Auto Hide Scrollbar,change Theme mode, set here your Theme Height mode,scroll bar Position mode,scroll Buttons mode,wheelSpeed,wheelPropagation,scroll Inertia & other setting from option panel.you can embed this scrollbar via setting panel.


= Plugin Features = 

* you can change height
* you can change Theme mode
* you can change atuo hide
* you can change scroll bar position mode
* you can change scroll buttons mode
* you can change wheel speed mode
* you can change wheel propagation mode
* you can change scroll Inertia



== Installation ==

This section describes how to install the sr-scrollbar-wp plugin and get it working.

It is nice and very easy to install,


1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'setting/scroll_bar_setting_manu' menu in WordPress
